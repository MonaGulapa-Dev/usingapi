First, let's define what an API is. An API, Application Programming Interface, is like a GUI for end user, only, it is a method of communication between software components. It is an application programming interface that lets your machine communicate with another machine, of course through software components.

REST stands for Representational State Transfer. It means when a RESTful API is called, the server will transfer to the client a representation of the state of the requested resource.

A RESTful API -- also referred to as a RESTful web service or REST API -- is based on representational state transfer (REST), an architectural style and approach to communications often used in web services development, a client-server architecture.

REST API provides a set of methods that a programmer or a developer can use via HTTP to send and receive data. HTTP requests are all we need to interact with RESTful API. Since these methods are over HTTP, any programming language can be used to employ REST API.

In this website, I used two related APIs - Google Map to represent the address with the map and IP locator which automatically detects your IP address and provides related information as to your location. You have to get the proper access from these API providers and read their documentation for the implementation.

As for a detailed explanation about REST API, I give kudos to this youtube by Clever Techie, a simple explanation that will give you a clear understanding of the concept.
